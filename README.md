# Skool.
Welcome to Skool. Where you go to school. In a unity game.
## Overall details
This is a school sandbox game where you can do the following:
* Chat with other students
* Go to class
* Skip class
* Explore the school
* Be a menace to society
* Escape school
* Be a teachers pet
* Be on your phone during school.
Basically, you can do anything that's not intended in school.
